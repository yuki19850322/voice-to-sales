# OpenAI Codex for Open Source 申請用 プロジェクトサマリー

## プロジェクト名
voice-to-sales

## キャッチフレーズ
「お客様の声」を貼るだけで、セールス文を3パターン自動生成するマイクロSaaS

## 解決する課題
- 良いレビューやお客様の声はあるのに、それを効果的なセールス文（LP・SNS投稿・メルマガ）に変換するのが非常に手間
- 特に日本語圏の営業マン、個人事業主、コーチ、講師にとって日常的に発生する痛み
- 既存の汎用AIでは「媒体に最適化されたトーン」で安定して出力しにくい

## ターゲットユーザー
日本語圏の
- 営業担当者
- 個人事業主・フリーランス
- コーチ・コンサルタント
- 講師・コンテンツクリエイター

## 主な機能
- 3種類のセールス文を同時生成
  - ビフォーアフター型（LP向き）
  - ストーリー型（メルマガ・ブログ向き）
  - 一言キャッチコピー（SNS・見出し向き）
- 媒体選択（LP / SNS投稿 / メルマガ）による文体自動最適化
- 1日3回までの無料利用制限（クライアントサイド実装）
- ワンクリックコピー機能

## 開発ストーリー（AIフル活用の好例）
1. **アイデア段階**：90個のマイクロSaaSアイデアから厳選（`ideas.md` / `おすすめベスト5.md`）
2. **設計段階**：Claude Code に依頼して詳細なMVP設計書を自動生成（`mvp-designs.md`）
3. **実装段階**：短期間で動作する完全なMVPを実装（フォーム＋Claude API＋3パターン出力＋制限＋UI）
4. **磨き・OSS化段階**：Grok Build を使用して再開し、以下の点を整備
   - 日本語で非常に充実した README
   - MIT License + .env.example
   - GitHub Actions による CI
   - 丁寧なデプロイガイド（DEPLOY.md）
   - モデルコメントの整理など軽微な polish

この一連のプロセス自体が「AIを活用した急速かつ質の高い開発」の実例となっています。

## 技術スタック
- **Frontend**: Next.js 16 (App Router) + React 19 + TypeScript + Tailwind CSS 4
- **AI**: Anthropic Claude API（`@anthropic-ai/sdk`、claude-3-5-sonnet-20241022）
- **プロンプトエンジニアリング**: 媒体別ガイド + 厳格なJSON出力強制 + 頑健な抽出ロジック
- **デプロイ**: Vercel（想定）
- **制限**: サーバーレス + クライアントサイド利用制限（最小構成）

## OSSとしての完成度（申請時点）
- ビジネス価値が明確な実用ツール
- 日本語ネイティブでターゲットが明確
- ドキュメントが充実（README + DEPLOY.md + 計画書群）
- ライセンス・CI・環境変数例が揃っている
- ライブデモ（Vercel）とソースコードの両方が公開

## 申請で特に強調したいポイント
1. **AIによる開発プロセスの透明性と再現性**  
   設計書生成から実装、ドキュメント整備まで一貫してAIを活用した記録が残っている。

2. **実際のビジネス課題を解決する実用性**  
   机上のツールではなく、営業・個人事業主が「今日から使える」レベルまで仕上げている。

3. **日本語圏特化のオープンソース事例**  
   英語圏中心のAIツールが多い中、日本語の営業シーンに特化した数少ない公開例。

4. **学習・再利用しやすい設計**  
   最小構成でありながら、プロンプトの工夫やデプロイ手順まで丁寧にドキュメント化。
